# Python: NumPy, Pandas, and Visualization

This repository contains a Jupyter Notebook that serves as a **hands-on tutorial** for mastering three essential tools in data science: **NumPy**, **Pandas**, and **Visualization (Matplotlib & Seaborn)**.  

The notebook is designed for learners and practitioners who want to strengthen their skills in **data manipulation, analysis, and visualization** with practical examples.

---

## 📚 Contents

### 🧮 NumPy
- Array creation (`np.array`, `np.ones`, `np.zeros`, `np.arange`, `np.linspace`, `np.eye`)
- Random number generation (`np.random.rand`, `np.random.randn`, `np.random.randint`)
- Array attributes (`.size`, `.shape`, `.ndim`, `.dtype`)
- Reshape vs Resize
- Indexing, slicing, and views vs copies
- Vectorized operations and matrix multiplication (`np.dot`)
- Stacking, concatenation, and splitting
- Broadcasting
- Utility functions (`np.where`, `np.select`, `np.clip`)

### 📊 Pandas
- Series and DataFrames creation
- Column and row operations (`.loc`, `.iloc`)
- Conditional selection and filtering
- Handling missing data (`isnull`, `dropna`, `fillna`)
- Merging and concatenation (`pd.merge`, `pd.concat`)
- GroupBy and aggregation (`.groupby()`, `.agg()`)
- Pivot tables and crosstabs
- Data inspection (`.head()`, `.tail()`, `.info()`, `.describe()`)
- Applying functions (`.map()`, `.apply()`)
- MultiIndex operations (`.stack()`, `.unstack()`)

### 📈 Visualization
- Line plots, scatter plots, bar plots, count plots
- Distribution plots (histogram, box plot, violin plot)
- Correlation plots (heatmap, pairplot)
- Subplots (`plt.subplots`)
- Customization (titles, labels, legends)
- Example visualizations using the Iris dataset
